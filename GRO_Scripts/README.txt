This scripts can be used by GRO application to generate simulations analyzed in the article.
--------------
For species with different dividing length, you can adjust division length by changing line 37 and line 40.
---------
Annulus.gro is used for generating inward growth.
--------
Symmetric.gro is used for outward growth.
--------

All the scripts gives output files, all the cell related datas in csv format, and bitmap images.
